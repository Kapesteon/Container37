# noVNC conteneur Docker

Ce projet permet de générer un client noVNC

## Installation

- Clonner le repository
- Rendez vous dans le dossier racine du projet et installez les dépendances :

```bash
npm i
# ou
yarn install
```

- Pour visualier le projet en local utilisez la commande :

```bash
npm run start
# ou
yarn start
```

- Pour générer des fichiers statiques prêts pour le déploiement utilisez la commande :

```bash
npm run build
# ou
yarn build
```

cela genèrera des fichiers .html .css et .js dans le dossier _dist_
